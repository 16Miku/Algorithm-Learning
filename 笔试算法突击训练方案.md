### 2. 数组/列表操作


## 三、核心算法模板（必背）

### 1. 滑动窗口模板

```python
def sliding_window(s):
    left = 0
    window = {}
    result = 0

    for right in range(len(s)):
        # 1. 扩大窗口
        c = s[right]
        window[c] = window.get(c, 0) + 1

        # 2. 收缩窗口
        while 窗口需要收缩:
            d = s[left]
            window[d] -= 1
            if window[d] == 0:
                del window[d]
            left += 1

        # 3. 更新结果
        result = max(result, right - left + 1)

    return result
```

### 2. 二分查找模板

```python
def binary_search(nums, target):
    left, right = 0, len(nums) - 1

    while left <= right:
        mid = left + (right - left) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1
```

### 3. BFS模板

```python
from collections import deque

def bfs(start):
    queue = deque([start])
    visited = {start}
    step = 0
```

### 4. DFS/回溯模板

```python
def backtrack(path, choices):
    if 满足结束条件:
        result.append(path[:])
        return

    for choice in choices:
        if 不满足条件:
            continue
        path.append(choice)
        backtrack(path, 新的选择列表)
        path.pop()
```

### 5. 动态规划模板

```python
def dp_solution(nums):
    n = len(nums)
    dp = [0] * n
    dp[0] = nums[0]

    for i in range(1, n):
        dp[i] = max(dp[i-1] + nums[i], nums[i])

    return max(dp)
```

### 6. 单调栈模板

```python
def monotonic_stack(nums):
    n = len(nums)
    result = [0] * n
    stack = []

    for i in range(n):
        while stack and nums[i] > nums[stack[-1]]:
            idx = stack.pop()
            result[idx] = i - idx
        stack.append(i)

    return result
```

## 五、笔试实战技巧

### 时间分配（3题120分钟）

- 题目1（简单）：25分钟
- 题目2（中等）：40分钟

### 做题策略

1. 先通读所有题目，标记难度
2. 先做会的题，保证基础分
3. 暴力解法也要写，能拿30-50%的分

| 坑点 | 解决方案 |
|------|----------|
| 数组越界 | 先判断边界条件 |
